## Features

- Login, register, manage personal information.
- Compose, edit, post blog to share with everyone.
- Like, save blogs on the read list so that you can read them Offline if needed.
- Localization

## Technology used

- Android XML for UI
- AndroidX
- Firebase Storage
- Dart Frog for backend
